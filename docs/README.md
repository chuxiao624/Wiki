# 插件介绍

#### 简介

SkyBlcok 空岛插件 , 针对基岩版服务器（BDS）的空岛插件 . 支持 LiteLoader 2.5.0 以上版本

>  正在筹备重构 (咕)


#### 基础功能

* **岛屿生成​**
* **岛屿范围保护​**
* **岛屿传送点设置​**
* **多人同岛**
* **自定义岛屿模板**
* **维度保护**
